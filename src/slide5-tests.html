Unit test of the loan computation

      
        describe("Loan calculator", () => {
          test("Test 1 year 5% rate", () => {
            const loanAmount = 1000;
            const durationMonths = 12;
            const anualInterestRate = 0.05;
            expect(computeLoan(loanAmount, durationMonths, anualInterestRate)).toEqual({
              monthlyRepayment: (1000 * (1 + 0.05)) / 12,
              totalOwned: 1000 * (1 + 0.05),
            });
          });
        });

Design principles in practice:

Testing is a very important part of a software developer work. In this project, I wrote a couple of unit test using the testing framework Jest.

  • Unit tests improve code maintainability.
  • It reduces errors and improve deliver quality.
  • Improves collaboration.

Other more powerful testing technique exits for web and App development. They allow integration testing e.g. Those based on browser automation.
