Design sketch

The initial sketch of the web application was designed with Figma.

  • It consists of two main part the form where the user inputs the loan amount and time, and stat bar.
  • The form contains two ways to input data a slider for quick and approximated values and a text box for precise control of the values.
  • The frontend is reactive and the user can quick see the effect of any change of the input on the final loan.
  • The color palet was chosen from Color Hunt. Tones of blue was chosen since blue color is normally assoiate to smart and technology.
